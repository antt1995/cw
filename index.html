WiFi

Show All Wifi Profiles
NETSH WLAN SHOW PROFILE *
NETSH WLAN SHOW PROFILE %%PROFILENAME%% KEY=CLEAR

User

Local User Managment

Add User
net user /add $USERNAME$ $PASSWORD$

With the password you need to double the First symbol aka %% for %

Add User to Local/Device Administrators
net localgroup administrators /add tetraadmin
net localgroup administrators AzureAD\$USERNAME$ /add

For AzureAD user

Reset Local Account password
net user $USERNAME$ $PASSWORD$

With the password you need to double the First symbol aka %% for %

Make Password never expire
WMIC USERACCOUNT WHERE Name='$USERNAME$' SET PasswordExpires=FALSE
Remove Local User
net user $USERNAME$ /delete

Bitlocker

Bitlocker Managment

Get bitlocker key/password
for /F "tokens=1,2" %%a in ('manage-bde -protectors -get C: -Type recoverypassword | findstr ID') do manage-bde -protectors -adbackup c: -id %%b
Get C: Protectors
manage-bde -protectors C: -get

Program's Managment

Get Installed ScreenConnect
wmic product where "Name like '%%ScreenConnect%%'" get name
Remove ScreenConnect
wmic product where name="ScreenConnect Client (******ID HERE*****)" call uninstall /nointeractive
